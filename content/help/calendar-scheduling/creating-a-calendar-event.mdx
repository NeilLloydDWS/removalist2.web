# Creating a calendar event

In addition to jobs, you can create custom events on your VanMan calendar. Use events for meetings, training sessions, vehicle servicing, or anything else that blocks out time.

<StepByStep>
  <Step title="Open the event dialog">
    On the calendar, click and drag to select a time range, or click the **New Event** button. The event dialog opens.
  </Step>

  <Step title="Enter event details">
    Give your event a **title** (e.g. "Team meeting" or "Truck 2 service"). Set the **start time** and **end time**, or toggle **All day** for events that span the entire day.
  </Step>

  <Step title="Choose a colour">
    Pick a colour for the event. This helps you distinguish events from jobs on the calendar. Each colour renders differently from the solid job blocks — events appear with dashed borders and a calendar icon.
  </Step>

  <Step title="Assign a truck (optional)">
    If the event is related to a specific truck (e.g. vehicle servicing), select it from the **Truck** dropdown. The event will appear in that truck's lane on the calendar.
  </Step>

  <Step title="Add contacts (optional)">
    Search for and add contacts to the event — useful for meetings with suppliers or customer appointments.

    <Callout variant="tip">
      Events assigned to a truck block that truck's availability for the duration. This prevents jobs from being scheduled during the event.
    </Callout>
  </Step>

  <Step title="Save the event">
    Click **Save**. The event appears on the calendar immediately, visible to all operators.
  </Step>
</StepByStep>

To edit or delete an event, click on it in the calendar and make your changes in the event dialog.
