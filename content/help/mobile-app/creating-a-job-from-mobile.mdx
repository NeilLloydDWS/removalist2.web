# Creating a job from mobile

Operators can create new jobs directly from the VanMan mobile app without returning to the office. The mobile job creation wizard guides you through three steps.

<StepByStep>
  <Step title="Step 1: General">
    Tap the **New job** button. Enter the customer's **name**, **phone number**, and **email address**. Set the **job date**, **start time**, and assign a **truck** from the dropdown. The truck list shows availability so you can avoid conflicts.
  </Step>

  <Step title="Step 2: Delivery">
    Enter the **pickup address** and **delivery address** using the address autocomplete. Add any additional stops if the move involves multiple locations. The autocomplete uses Google Maps to verify addresses and enable navigation later.
  </Step>

  <Step title="Step 3: Tally">
    Add the job charges and line items. Enter the pricing breakdown for the move, including hourly rates, fixed charges, or any additional fees. This information is used when generating invoices.
  </Step>
</StepByStep>

Tap **Save job** to create the job. It immediately appears on the calendar and is visible to all team members on both mobile and web.

<Callout variant="tip">
  Use mobile job creation when a customer calls while you are out in the field. You can take the booking, assign a truck, and have it on the calendar before the call ends.
</Callout>

<Callout variant="info">
  The mobile wizard collects the same information as the web job form. Any job created on mobile is fully editable from the web dashboard.
</Callout>
