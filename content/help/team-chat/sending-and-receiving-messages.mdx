# Sending and receiving messages

Team chat in VanMan lets you exchange text messages with your crew members in real time. Messages sync across web and mobile so you can continue a conversation from either device.

## Sending a message

Open a conversation from the <UIPath path="Chat" /> screen and type your message in the input field at the bottom. Tap **Send** to deliver it. Messages can be up to 2,000 characters long.

<Callout variant="tip">
  Keep messages concise and actionable. Chat works best for quick coordination — "Customer wants a 10am start instead of 9am" — rather than long detailed instructions. Use job notes for detailed information your crew needs on-site.
</Callout>

## Receiving messages

When a team member sends you a message, it appears in the conversation in real time. New messages are shown with a timestamp and the sender's name. If you are not currently viewing the conversation, an unread badge appears on the chat list.

## Message history

All messages are stored and accessible at any time. Scroll up in a conversation to see earlier messages. Your entire message history with each team member is preserved so you can reference past discussions.

<Callout variant="info">
  Messages sync between web and mobile in real time. A message sent from the mobile app appears on the web dashboard immediately, and vice versa. You can start a conversation on one device and continue it on another.
</Callout>
